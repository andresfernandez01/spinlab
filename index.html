YOUTUBE DATA API v3 KEY
Required for track search. Get a free key at console.cloud.google.com → APIs & Services → Credentials.
HOW TO GET A KEY
  1. Go to console.cloud.google.com
  2. Create or select a project
  3. APIs & Services → Library → enable YouTube Data API v3
  4. Credentials → Create Credentials → API Key
  5. Paste it above and click Save
Your key is stored locally in your browser only — never sent anywhere except YouTube's API.
